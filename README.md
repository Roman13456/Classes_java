## Strings in Java

**Clone the repository to your local machine**
```bash
git clone https://github.com/Roman13456/Classes_java.git
```

**Navigate to the project directory**
```bash
cd Classes_java
```

**Compile the code (if not using an IDE)**
```bash
javac -d bin src/classes.java
```

**Run the program**
```
java -cp bin classes
```
